# ☕ Product Card UI

A clean, minimal **e-commerce product card** built with pure HTML & CSS — no frameworks, no libraries.

## 🎨 Features

- ✅ Responsive product card layout
- ✅ Original price with strikethrough & discounted price
- ✅ Hover effects on Add to Cart button
- ✅ Clean typography using **DM Serif Display** (Google Fonts)
- ✅ Subtle box shadow & border radius for modern look
- ✅ Pure HTML & CSS — zero dependencies

## 🏃 How to Run

1. **Clone the repo:**
   ```bash
   git clone https://github.com/AI-RISHVA/product-card.git
   ```

2. **Open in browser:**
   ```bash
   cd product-card
   open index.html
   ```
   Ya simply `index.html` ko browser mein drag & drop karo!
